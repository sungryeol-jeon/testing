🚫 Pipeline FAIL — Merge 차단

  • Filelist: MISSING / NOT FOUND 패턴
  • Lint: Spyglass Error 패턴 (^Error, ^E-)
  • Compile: VCS Error-[, **Error 패턴
  • Simulate: FAIL / TIMEOUT / ERROR 패턴
  • Synthesis: DC Error 패턴

⚠️ WARN — Merge 허용 (allow_failure)

  • Lint: Warning만 존재 (^Warning, ^W-)
